Using Data Integration Tools with Autonomous Data Warehouse
  • Using Data Integration Tools with Autonomous Data Warehouse
  • Getting Started with DIPC and ADW (flatfile to AWD)
  • Getting Started with DIPC and ADW (Oracle Database to ADW replication)
 In these labs, you will configure and use Oracle Data Integration Platform Cloud (DIPC) with Autonomous Data Warehouse. This lab follow a typical enterprise data warehouse reference implementation with ETL/ELT batch processing, real time data replication, and data quality review. You will load data from a flat file and a database table using Oracle Data Integrator (ODI) to your Autonomous Data Warehouse. You will replicate data from a database table to Autonomous Data Warehouse using Oracle Golden Gate (OGG). You will review data quality in your Autonomous Data Warehouse using Oracle Enterprise Data Quality (EDQ).

Autonomous Solution Workshops First Success Labs to Position with Customers Overview
Demand Prediction Setup Workshop
  • ML Data Preparation Using SQL Developer
  • Getting Started with Oracle Analytics Cloud (OAC)
 This workshop walks you through using the Oracle Autonomous Data Warehouse (ADW) for storing analytical datasets and applying Machine Learning models to learn the correlation between attributes and predict future demand.

You will take on 2 personas during the workshop: the Data Scientist prepares the data, validates the machine learning models, and applies those models to predict future demand; the Business Analyst also builds and applies Machine Learning models using Oracle Analytics Cloud service (OAC) without writing a single line of code. During the workshop you will work with Oracle Autonomous Data Warehouse, Oracle Machine Learning tool, and Oracle Analytics Cloud Service.
Policy Based Scaling Workshop This workshop walks you through provisioning the Oracle Autonomous Data Warehouse (ADW), creating Database as a Service instance, loading data, monitoring and management of ADW, auto scaling up and down of an ADW instance, and policy setting for ADW scaling.
SaaS Optimized Reporting Workshop This workshop walks you through how to utilize Oracle Analytics Cloud (OAC) to extend analysis of project data from Oracle Enterprise Resource Planning (ERP). You will learn how to pull in curated content using the tight integration between OAC and ERP; blend essential third-party data staged in Autonomous Data Warehouse; perform what-if analysis and apply machine learning to create intuitive visualizations to help business stakeholders make swift and informed decisions.
